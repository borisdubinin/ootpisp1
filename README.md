# GraphEditor

> Если планируете использовать эту версию программы для сдачи лабы, то попрошу поменять радикально интерфейс, чтобы не было схожести.
> Также перед клонированием для собственного удобства сделайте Fork, в котором будете работать, чтобы сохранять свои изменения в удаленный репозиторий.

## Краткая инструкция по использованию

### Инструменты (Горячие клавиши)
- **V** — Режим выделения курсором
- **R** — Прямоугольник
- **T** — Треугольник
- **H** — Шестиугольник
- **D** — Ромб
- **Z** — Трапеция
- **C** — Круг

### Управление и функции
- **ЛКМ (Левая кнопка мыши)** — Выделить фигуру, потянуть для перемещения. Двойной клик на фигуру включает режим редактирования вершин.
- **ПКМ (Правая кнопка мыши)** — Открыть контекстное меню для создания новой фигуры или параметров.
- **Средняя кнопка мыши** или **Пробел + ЛКМ** — Перемещение камеры (Pan).
- **Колесико мыши** или **Ctrl + `+`/`-`** — Масштабирование камеры (Zoom). **Ctrl + `0`** для сброса.
- **Стрелки** — Точное перемещение фигуры (+ **Shift** для шага x10).
- **Delete / Backspace** — Удалить выделенную фигуру.
- **Escape** — Снять выделение / Отменить действие / Выход из режима редактирования.
- **N** — Включить / выключить режим редактирования вершин.
- **G** / **A** — Показать / скрыть сетку и оси.

### Трансформация фигур
- **Масштабирование**: Потяните за маркеры рамки выделения (для пропорционального изменения удерживайте **Shift** за угловой маркер).
- **Вращение**: Потяните за верхний круглый маркер (удерживайте **Shift** для привязки к углу 15°).
- **Перемещение якоря**: Удерживайте **Alt** и перетащите маркер якоря.

## Описание сборки проекта

Данный проект использует систему сборки CMake и автоматически скачивает все необходимые зависимости (SFML 2.6.2, ImGui, ImGui-SFML) на этапе конфигурации с помощью `FetchContent`. Вам не требуется предварительно скачивать и устанавливать эти библиотеки.

### Требования к системе

Для успешной компиляции на вашем компьютере должны быть установлены:
- **CMake** (версия 3.20 или выше).
- **Компилятор C++** с поддержкой стандарта **C++17** (GCC, MacOS Clang, Apple Clang или MSVC).

#### Дополнительные зависимости (только для Linux)
На ОС Linux (например, Ubuntu/Debian) для сборки библиотеки SFML из исходных кодов могут потребоваться дополнительные системные пакеты. Установите их перед сборкой:
```bash
sudo apt-get update
sudo apt-get install cmake gcc g++ libxrandr-dev libxcursor-dev libudev-dev libfreetype-dev libgl1-mesa-dev libflac-dev libogg-dev libvorbis-dev libopenal-dev
```

На Mac (с установленным Xcode Command Line Tools) и Windows (с Visual Studio) дополнительные зависимости не требуются.

### Пошаговая инструкция по сборке

1. **Склонируйте репозиторий** и перейдите в его директорию:
   ```bash
   git clone <URL_репозитория>
   cd <название_папки_репозитория>
   ```

2. **Создайте папку для сборки** (чтобы не засорять корневую директорию проекта) и перейдите в нее:
   ```bash
   mkdir build
   cd build
   ```

3. **Сгенерируйте файлы сборки** с помощью CMake:
   ```bash
   cmake ..
   ```
   *Примечание: При первом запуске этот процесс может занять некоторое время, так как CMake будет скачивать исходные коды SFML и ImGui из интернета.*

4. **Скомпилируйте проект**:
   ```bash
   cmake --build .
   ```
   *Для ускорения компиляции на многоядерных процессорах можно использовать флаг параллельной сборки. Например, для 8 потоков: `cmake --build . -j8`*

   *На Windows при использовании Visual Studio можно дополнительно указать конфигурацию (по умолчанию собирается Debug):* 
   `cmake --build . --config Release`

### Запуск скомпилированного приложения

После успешного завершения компиляции, исполняемый файл появится в папке сборки. 

Для запуска используйте:
- **На macOS / Linux:**
  ```bash
  ./GraphEditor
  ```
- **На Windows:**
  В зависимости от генератора, файл может лежать в подпапке (например, `Debug` или `Release`). Запуск через терминал:
  ```cmd
  Debug\GraphEditor.exe
  ```

### Возможные проблемы
Если CMake не может найти компилятор на Windows, убедитесь, что вы производите сборку из-под "Developer Command Prompt for VS" или запустили CMake GUI и выбрали правильный генератор Visual Studio.
На macOS убедитесь, что у вас установлены инструменты разработчика командой `xcode-select --install`.
